# Luxury Real Estate HTML Email Template

This repository contains a **responsive, email-safe HTML email template**
designed for **luxury real estate marketing**.

## Features
- Email-client compatible (Gmail, Outlook, Apple Mail)
- Responsive for mobile, tablet, and desktop
- Table-based layout with inline styles
- Supports dynamic content via JSON
- Ready for ChatGPT + n8n automation workflows

## Usage
- The HTML file is the **final email output**
- Dynamic values are injected using placeholders (e.g. {{client_name}})
- JSON data is generated by ChatGPT and injected using n8n before sending

## Preview
GitHub preview is for **visual reference only**.  
Actual email rendering may vary by email client.

## Technologies
- HTML (email-safe)
- Inline CSS
- JSON (dynamic content)
- n8n (automation)
